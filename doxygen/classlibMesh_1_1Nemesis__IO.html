Detailed Description

The Nemesis_IO class implements reading parallel meshes in the Nemesis file format from Sandia National Labs. Nemesis files are essentially in the Exodus format plus some additional information. All the Nemesis files for a single mesh have the same basename, e.g. cylinder.e, followed by ".size.rank", where size is the total number of files the Mesh is split into and rank is the ID of the processor's elements that were written to the file.

John Peterson, 2008.

526  // We know we will receive the request from a given processor before
527  // we receive its reply to our request. However, we may receive
528  // a request and a response from one processor before getting
529  // a request from another processor. So what we are doing here
530  // is processing whatever message comes next, while recognizing
531  // we will receive a request from a processor before receiving
532  // its reply
533  std::vector<bool> processed_cmap (nemhelper->num_node_cmaps, false);
534 
535  for (unsigned int comm_step=0; comm_step<2*to_uint(nemhelper->num_node_cmaps); comm_step++)

537  // query the first message which is available
538  const Parallel::Status

540  nodes_tag));

557

559  // avoid repeated map lookups
560  std::vector<int> &xfer_buf (requested_node_idxs[requesting_pid_idx]);
561 
562  // actually receive the message.
563  this->comm().receive (requesting_pid_idx, xfer_buf, nodes_tag);
564 
565  // Fill the request
566  for (unsigned int i=0; i<xfer_buf.size(); i++)
567  {

569  const unsigned int old_global_node_idx = xfer_buf[i];

572  // note that requesting_pid_idx thinks we own this node,
573  // so we better!

575  std::lower_bound (old_global_to_new_global_map.begin(),
576  old_global_to_new_global_map.end(),
577  old_global_node_idx,
578  global_idx_mapping_comp);

582  libmesh_assert_greater_equal (it->second, my_node_offset);
583  libmesh_assert_less (it->second, my_next_node);
584

586  xfer_buf[i] = it->second;
587  }

590  this->comm().send (requesting_pid_idx,
591  xfer_buf,
592  requested_nodes_requests[cmap],
593  nodes_tag);
594  } // done processing the request

597  // so it must be its reply to our request

613  nodes_tag);
